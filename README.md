# Akkariin for YSU

> You cannot see this repository because it is transparent.

仅供学习 Java Web 使用，作者不对使用本项目代码造成的任何损失负责。

理论支持的平台：Desktop (JVM), Android （手头没有苹果设备所以不支持 iOS）

## 功能

如果你有想要的功能可以提 issue，最好带着逆向好的 API

- ✅ 校园卡余额查询
- ❌ 绩点查询
- ❌ 课表
- ❌ 图书馆空闲查询/预约
- ❌ 社团推荐
